## Default Permission

Default permissions for tauri-plugin-litert. Allows model management and inference.

#### This default permission set includes the following:

- `allow-load-model`
- `allow-unload-model`
- `allow-list-models`
- `allow-get-model-info`
- `allow-run-inference`
- `allow-create-embedding`
- `allow-load-lm-model`
- `allow-unload-lm-model`
- `allow-list-lm-models`
- `allow-generate`
- `allow-generate-stream`
